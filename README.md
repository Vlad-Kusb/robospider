### Режимы работы робота (переключение кнопкой)

| Количество нажатий | Режим | Описание |
|:---:|:---|:---|
| 1 | Стартовая позиция | Возврат в исходное положение |
| 2 | Демонстрация (танец) | Исполнение танцевальных движений |
| 3 | Ходьба | Движение вперёд / шаги |
| 4 | Сон | Выключение или низкое энергопотребление |
